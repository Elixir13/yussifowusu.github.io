Surveillance of Antimicrobial Resistance Patterns in Clinical Isolates

Analysis of antimicrobial susceptibility data from hospital isolates to monitor resistance trends, inform treatment guidelines, and develop interactive surveillance dashboards.

  • Power BI
  • Excel
  • Python
  • SPSS
  • WHONET

Project Overview

This project involved the systematic analysis of antimicrobial resistance (AMR) patterns among bacterial isolates from clinical samples. Data visualization tools were employed to create dashboards for real-time monitoring and to support evidence-based antimicrobial stewardship programs.

Methodology

Susceptibility testing data were processed using WHONET software and analyzed with SPSS and Python. Interactive dashboards were built in Power BI to facilitate trend analysis and reporting.

Results & Visualizations

Key findings included rising resistance to common antibiotics in Gram-negative pathogens. Forest plots and trend charts highlighted priority areas for intervention.

Outcomes & Impact

The surveillance system supported hospital infection control policies and contributed to national AMR reporting efforts, promoting rational antibiotic use.
